# 6. Merge Two Sorted Lists
**Difficulty**: Easy

Merge two sorted linked lists and return it as a new sorted list.

### **Example**
Input: list1 = [1, 2, 4], list2 = [1, 3, 4]  
Output: [1, 1, 2, 3, 4, 4]  

### **Constraints**
- The number of nodes in both lists is in the range [0, 50].
- -100 <= Node.val <= 100
