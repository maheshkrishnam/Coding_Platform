### Problem 2: Reverse a String

#### Description:
Given a string `s`, reverse it and print the resulting string. You are required to implement a function to reverse the string for each test case.

#### Input:
- An integer `t` (1 ≤ t ≤ 100) representing the number of test cases.
- For each test case, a string `s` of length `n` (1 ≤ n ≤ 10^5).

#### Output:
- For each test case, output the reversed string.

#### Example:
**Input:**
- 3 hello world codeforces

**Output:**
- olleh dlrow secrofedoc

#### Notes:
- The string consists only of lowercase English letters.
- The solution should handle large input sizes efficiently.
